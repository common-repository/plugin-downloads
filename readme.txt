== Description ==

Easily show your plugin download statistics on your blog in a cool sortable table.

Usage: type [plugin_downloads] within a post or page.

More info:

* [Plugin Downloads](http://yoast.com/wordpress/plugin-downloads/).
* Check out the other [Wordpress plugins](http://yoast.com/wordpress/) by the same author.

== Installation ==

Installation is easy:

* Download and unzip the plugin.
* Copy the folder to the plugins directory of your blog.
* Enable the plugin in your admin panel.
* An options panel will appear under Settings.
